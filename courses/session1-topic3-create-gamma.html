Task: Create a one-pager for an upcoming event at your institution.

Prompt Template:

Create a professional one-pager for [EVENT NAME]. Event Details: - Date: [DATE] - Time: [TIME] - Location: [LOCATION] - Target Audience: [WHO SHOULD ATTEND] Include: - Eye-catching title - Event description (2-3 sentences) - Key benefits of attending - Registration information - Contact details Style: [professional/casual/modern] with [COLOR SCHEME] colors.

What to Include:

  • Event title and tagline
  • Date, time, and location
  • Brief description
  • Key benefits or learning outcomes
  • How to register or RSVP
  • Contact information
